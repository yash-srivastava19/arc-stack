# Contributing to arc

## Dev setup

```bash
uv sync                # install deps
pip install -e .       # editable install — required so `arc --version` and
                       # importlib.metadata resolve (run after every uv sync)
git config core.hooksPath .githooks   # run ruff/mypy/pytest before each commit
```

## Running checks

```bash
uv run pytest -x -q                                  # full suite
uv run pytest -m unit                                # pure logic, no I/O
uv run pytest -m git                                 # needs a real local git repo
uv run pytest -m api                                 # vcrpy cassettes (GitHub API)
uv run ruff check arc tests && uv run ruff format --check arc tests
uv run mypy arc
```

CI enforces lint, types, and an 80% coverage gate. Never merge red.

## Code layout

- `arc/cli.py` — Click group + global flags only; registers commands from `arc/commands/`
- `arc/commands/` — one module per command family (`stack`, `submit`, `sync`, `config`, `diagnostics`, `nav`); `_shared.py` holds cross-command helpers
- `arc/hooks.py` — generic lifecycle hook runner; **stdlib imports only** (enforced by test); designed for extraction
- `arc/{git,github}.py` — subprocess wrappers around `git`/`gh`; commands import these as modules so tests can patch `arc.git.*`
- `arc/{state,ops,graph,conflicts}.py` — pure logic, no I/O beyond state files

## Conventions

- Conventional commits (`feat:`, `fix:`, `refactor:`, `test:`, `docs:`, `release:`, `chore:`, `style:`, `infra:`, `ci:`, `qol:`, `recover:`) — CHANGELOG is generated by git-cliff from these types (see `cliff.toml`'s `commit_parsers`). A squash-merge uses the PR title as the commit message, so **the PR title itself must follow this format** — CI checks it (`.github/workflows/pr-title.yml`)
- Errors follow `error: <what>` + `hint: <exact fix>` (two lines, hint dim)
- New commands: support `--json` where output is data, `-q/--quiet`, and respect `--no-input`
- Multi-commit work goes through arc-managed stacked PRs (dogfooding), not raw `gh pr create`
